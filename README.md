# P5R SML

Persona 5 Royal Simple Mod Loader is a quick alternative to Reloaded-II. It supports packed files and editing `.SPD`, `.PAC`, and `.BIN` files on the fly.

This mod loader is not intended to replace Reloaded-II, instead it was made as an effort to just play Persona 5 with a few mods on Linux without having to go through the hassle of installing Reloaded-II via Proton.

I might try my hand at implementing my own versions of some frameworks, but that would be a heavy undertaking due to them being written in C# necessitating a full rewrite basically.

I do have Persona 4 Golden support, but it is not tested much beyond simple `.PAC` mods.
